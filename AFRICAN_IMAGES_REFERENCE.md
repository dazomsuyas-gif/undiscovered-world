## 📝 HOW TO REPLACE IMAGES

### **Step 1: Find Image in HTML**

Open `index.html`, search for:
```html
<img src="https://images.unsplash.com/photo-1507525428034-b723cf961d3e
```

### **Step 2: Replace URL**

Change from:
```html
<img src="https://images.unsplash.com/photo-1507525428034-b723cf961d3e?w=400&h=300&fit=crop"
     alt="Beach">
```

To:
```html
<img src="https://images.unsplash.com/photo-1516426122078-c23e76319801?w=400&h=300&fit=crop"
     alt="Zanzibar Beaches - Tanzania">
```

### **Step 3: Update Alt Text**

**Old:** `alt="Beach Destination"`
**New:** `alt="Zanzibar Beaches, Tanzania"`

**Old:** `alt="Mountain Adventure"`
**New:** `alt="Mount Kilimanjaro, Tanzania"`

### **Step 4: Save & Deploy**

```bash
git add .
git commit -m "Updated images to African/Tanzania destinations"
git push origin main
```
