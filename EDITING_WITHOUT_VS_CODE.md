## ⚡ **5-MINUTE SETUP**

### **What You'll Do:**

1. **Add blog link** (1 min)
   - Open index.html in Notepad
   - Add: `<li><a href="blog.html">Blog</a></li>`
   - Save

2. **Push to GitHub** (2 min)
   ```bash
   git add .
   git commit -m "Added blog page"
   git push origin main
   ```

3. **Start posting** (2 min)
   - Go to: `yoursite.netlify.app/blog.html`
   - Fill form
   - Click "Post Now"

**Done!** ✅
