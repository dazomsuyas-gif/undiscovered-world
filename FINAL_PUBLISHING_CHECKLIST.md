## ❓ **IF SOMETHING GOES WRONG**

### **"Permission denied"**
→ Check authentication (password/token correct)

### **"fatal: not a git repository"**
→ Folder not initialized, run: `git init`

### **"rejected - non-fast-forward"**
→ Run: `git pull origin main` first

### **"Nothing to commit"**
→ No changes detected, all files already uploaded

### **Website shows blank**
→ Make sure index.html is in root folder
→ Hard refresh: Ctrl+Shift+R
