### **URGENT (Do This Week)**

**Task 1: Optimize Website for SEO**
```
Time: 2-3 hours
Steps:
1. Open index.html in text editor
2. Find meta tags (around line 6-12)
3. Update meta description with keywords
4. Do same for about.html and contact.html
5. Add more content to each section
6. Save files
7. Upload to Netlify
```

**Task 2: Set Up Google Search Console**
```
Time: 20 minutes
Steps:
1. Go to: https://search.google.com/search-console
2. Add your website URL
3. Verify ownership (add HTML tag)
4. Submit sitemap
5. Check coverage report
```

**Task 3: Set Up Google Analytics**
```
Time: 15 minutes
Steps:
1. Go to: https://analytics.google.com
2. Create account
3. Add your website
4. Get tracking code
5. Add to your website <head>
6. Wait 24 hours for data
```
